# Machine-Learning-Lab-Codes

Complete Python implementations of Machine Learning Lab experiments with NumPy, Pandas, Statistics, Regression, and Classification models.

## Repository Structure

Each lab contains 8 experiments (Exp1 through Exp8) implemented as Jupyter notebooks.

### Lab 01: Python, NumPy & Pandas Basics
- Exp1: NumPy array creation
- Exp2: Matrix operations
- Exp3: Random arrays
- Exp4: Pandas DataFrames
- Exp5: Load CSV files
- Exp6: EDA - head(), tail(), info()
- Exp7: EDA - describe(), dtypes
- Exp8: Data type conversions

### Lab 03: Descriptive Statistics
- Exp1-Exp2: Mean, median, mode
- Exp3: Variance and standard deviation
- Exp4: Quartiles and percentiles
- Exp5: Correlation analysis
- Exp6: Covariance matrix
- Exp7: Histograms
- Exp8: Boxplots

### Lab 04: Regression & Classification
- Exp1: Simple Linear Regression
- Exp2: Multiple Linear Regression
- Exp3: Regression metrics (RMSE, R², MAE, MSE)
- Exp4: Logistic Regression
- Exp5: Cross-validation
- Exp6: Classification metrics & Confusion Matrix
- Exp7: ROC Curve
- Exp8: Precision-Recall Curve

## Requirements

```
python >= 3.7
jupyter
numpy
pandas
scikit-learn
matplotlib
seaborn
```

## Installation

```bash
pip install -r requirements.txt
jupyter notebook
```

## Usage

1. Navigate to any lab folder
2. Open Jupyter notebook files (Exp1.ipynb, Exp2.ipynb, etc.)
3. Run cells sequentially
4. Modify and experiment with the code
